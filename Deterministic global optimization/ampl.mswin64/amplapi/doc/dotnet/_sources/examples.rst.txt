C# API Examples
===============

This section lists a few examples in C#. 
These are the same files that can be found in the *examples* directory of the distribution, and show the basic usage of the C# API. 

Example 1: First steps
----------------------

:download:`firstexample.cs <../../cpp/swig/csharp/examples/FirstExample.cs>`

This example shows how to
 
* read an AMPL model
* reassign values to parameters 
* solve the model 
* display the objective function value

Example 2: Get and set AMPL options
-----------------------------------

:download:`optionsexample.cs <../../cpp/swig/csharp/examples/OptionsExample.cs>`

This example shows how to:

* get and set AMPL options
   
Example 3: Get and set data through DataFrames
----------------------------------------------

:download:`dataframeexample.cs <../../cpp/swig/csharp/examples/DataFrameExample.cs>`

This example shows how to:

* Use :cs:class:`DataFrame` objects to assign values to sets and parameters
* Use an :cs:class:`DataFrame` object to fetch and decouple values from a variable
   
Example 4: Operate with multidimensional data
---------------------------------------------

:download:`multidimensionalexample.cs <../../cpp/swig/csharp/examples/MultidimensionalExample.cs>`

This example shows how to:

* Use a :cs:class:`DataFrame` to assign values to multidimensional parameters

--------------------------------------------------

:download:`dietmodel.cs <../../cpp/swig/csharp/examples/DietModel.cs>`
